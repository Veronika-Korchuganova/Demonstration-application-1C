## 📖 О проекте
Конфигурация автоматизирует базовый процесс оптовой торговли: от создания карточки товара до фиксации его продажи и списания со склада.
* **Цель:** Изучить подходы к качественной разработке и автоматизации тестирования в 1С.
* **Основные подсистемы:** Справочники (номенклатура и услуги), Документы (заказ, поступление, продажа)

## 🛠 Технологический стек
* **Платформа:** 1С:Демонстрационное приложение (1.0.39.2)
* **Инструменты тестирования:** Vanessa Automation, Сценарное тестирование 1С
* **Инфраструктура:** OneScript, GitHub Actions (или GitLab CI)
* **Отчетность:** Allure Report

## 🧪 Автотесты
Автоматизация тестирования реализована по методологии **BDD (Behavior-Driven Development)** на базе **Vanessa Automation**.

* **Сценарии на языке Gherkin:** Все файлы `.feature` расположены в каталоге `/tests/features`.
* **Покрытие:** Автотестами покрыты критические цепочки (например, создание номенклатуры -> создание контрагента -> оформление заказа).

### Пример фичи (ЗаказПокупателя.feature):
```gherkin
Функционал: Создание и проведение документа "Продажа"

Сценарий: Создание и проверка документа продажи
	*Создание продажи
		И В командном интерфейсе я выбираю "Продажи" "Продажи"
		Тогда открылось окно "Продажи товара"
		И я нажимаю на кнопку с именем 'ФормаСоздать'
		Тогда открылось окно "Продажа товара (создание)"
		И я нажимаю кнопку выбора у поля с именем 'Организация'
		И из выпадающего списка с именем 'Организация' я выбираю по строке "ВА - Организация"
		И из выпадающего списка с именем 'Покупатель' я выбираю по строке "ВА - задание"
		И я нажимаю кнопку выбора у поля с именем 'Склад'
		И из выпадающего списка с именем 'Склад' я выбираю по строке "ВА - Основной"
		И в таблице 'Товары' я нажимаю на кнопку с именем 'ТоварыДобавить'
		И в таблице 'Товары' из выпадающего списка с именем 'ТоварыТовар' я выбираю по строке "ВА Товар"
		И в таблице 'Товары' я активизирую поле с именем 'ТоварыЦена'
		И в таблице 'Товары' в поле с именем 'ТоварыЦена' я ввожу текст "100,00"
		И в таблице 'Товары' я завершаю редактирование строки
		И я нажимаю на кнопку с именем 'ФормаПровести'	
		И я запоминаю значение поля с именем "Номер" как "НомерСозданияПродажи"
		И я запоминаю значение поля с именем "Дата" как "ДатаСозданияПродажи"
	*Проверка создания продажи
		И В командном интерфейсе я выбираю "Продажи" "Продажи"
		Тогда открылось окно "Продажи товара"
		И таблица "Список" содержит строки:
			| 'Дата'                  | 'Номер'                  |
			| '$ДатаСозданияПродажи$' | '$НомерСозданияПродажи$' |
```
